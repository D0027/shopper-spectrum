## 🧠 How It Works

```
Raw Transaction Data (541,909 rows)
         │
         ▼
   Data Cleaning & Feature Engineering
         │
         ├──► RFM Computation → KMeans (k=4) → Customer Segments
         │         High-Value │ Regular │ Occasional │ At-Risk
         │
         └──► Product-Customer Matrix → Cosine Similarity → Recommendations
```

### Segmentation Logic (RFM + KMeans)
- **Recency** — Days since last purchase
- **Frequency** — Number of unique orders
- **Monetary** — Total spend (£)
- Features log-transformed + StandardScaled before clustering

### Recommendation Engine
- Built a **Customer × Product** purchase matrix
- Computed **Item-Based Cosine Similarity** across all 2,941 products
- Returns top-N most similar products for any query product

## 🚀 Quick Setup (3 Steps)

### Prerequisites
- Python 3.8+
- pip

### Step 1 — Install dependencies

```bash
pip install -r requirements.txt
```

### Step 2 — Train the models

```bash
python train_models.py
```

> Reads `data/online_retail.csv`, cleans data, trains KMeans + builds cosine similarity matrix, saves everything to `models/`. Takes ~1–2 minutes.

### Step 3 — Launch the app

```bash
streamlit run app.py
```

Open your browser at **http://localhost:8501** 🎉
